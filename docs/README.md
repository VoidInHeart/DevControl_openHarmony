## 阅读约定

- 当前所有设备均为 `VirtualGateway` 内的软件模拟驱动，不代表已经兼容真实家电厂商、Matter、Zigbee、蓝牙或私有硬件协议。
- 二维码中的设备身份证书是静态、可长期保存的 JWS；APP 与网关的配对会话仍会在网关重启或 24 小时凭据到期后失效，两者不能混淆。
- 文档中的令牌、私钥、签名密码仅以占位符表示，不能记录在文档、二维码命令历史或提交内容中。
